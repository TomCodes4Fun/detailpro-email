# Car Detailing CRM - Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1e293b) - Professional, trustworthy, automotive industry standard
- **Secondary**: Soft Blue (#3b82f6) - Modern, clean, tech-forward
- **Accent**: Warm Orange (#f97316) - Energy, action, call-to-action elements
- **Neutral**: Cool Gray (#64748b) - Text, borders, subtle elements
- **Background**: Pure White (#ffffff) - Clean, professional, high contrast
- **Success**: Emerald (#10b981) - Completed tasks, positive status
- **Warning**: Amber (#f59e0b) - Pending items, attention needed

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings and brand elements
- **Body Font**: "Inter" - Clean, highly readable sans-serif for all interface text
- **Monospace**: "JetBrains Mono" - For data, numbers, and technical information

### Visual Language
- **Minimalist Approach**: Clean lines, ample whitespace, focused content
- **Professional Aesthetic**: Business-appropriate with subtle automotive references
- **Mobile-First**: Touch-friendly elements, thumb-accessible navigation
- **Data-Driven**: Clear hierarchy, scannable information architecture

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions and state transitions
- **ECharts.js**: Professional data visualization for business metrics
- **Splide**: Image carousels for service galleries
- **p5.js**: Subtle background animations and visual elements
- **Pixi.js**: High-performance visual effects for dashboard

### Animation Strategy
- **Micro-interactions**: Button hover states, form field focus, loading indicators
- **Page Transitions**: Smooth slide animations between sections
- **Data Visualization**: Animated chart updates and progress indicators
- **Loading States**: Skeleton screens and progressive content loading

### Header Effects
- **Subtle Gradient Flow**: Animated background using CSS gradients
- **Floating Elements**: Gentle movement of automotive-themed icons
- **Parallax Scrolling**: Depth effect on hero sections (minimal, mobile-optimized)

### Interactive Elements
- **Touch Feedback**: Immediate visual response to user interactions
- **Progressive Disclosure**: Information revealed based on user actions
- **Contextual Animations**: Visual feedback for form validation and status changes
- **Gesture Support**: Swipe navigation for mobile galleries and calendars

### Mobile Optimization
- **Thumb-Friendly**: All interactive elements sized for easy touch interaction
- **One-Handed Use**: Critical functions accessible within thumb reach
- **Swipe Gestures**: Natural mobile navigation patterns
- **Haptic Feedback**: Visual indicators for touch interactions

### Background Treatment
- **Consistent Base**: Solid white background throughout application
- **Subtle Texture**: Very light geometric patterns in header areas only
- **No Gradients**: Maintaining clean, professional appearance
- **Visual Separation**: Using shadows and borders instead of color changes

### Component Styling
- **Cards**: Subtle shadows, rounded corners (8px radius)
- **Buttons**: Solid colors, clear hierarchy, adequate touch targets (44px minimum)
- **Forms**: Clean inputs with focus states, inline validation
- **Navigation**: Bottom-positioned for mobile, clear visual hierarchy
- **Data Tables**: Alternating row colors, sortable headers, mobile-responsive
